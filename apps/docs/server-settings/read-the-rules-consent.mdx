## What it does

Marks new users to your server as consenting to [display their messages](/user-settings/displaying-messages) publicly on Answer Overflow.


<Warning>
	Make sure you have the following in your [membership
	screening](https://support.discord.com/hc/en-us/articles/1500000466882-Rules-Screening-FAQ)
	
	`Some channels on this server are indexed by Answer Overflow. By posting in these channels, you agree to have your messages publicly displayed on the web.`
</Warning>
