---
title: Ignore Account in Server
description: Opt out of Answer Overflow for a specific server
---

Use `/manage-account` and click **Ignore account in server** to opt out of indexing for just one server while staying visible in others.

<Warning>
This will:
- Delete all your stored messages in that server
- Stop indexing future messages in that server
- Stop tracking analytics for your account in that server
</Warning>

## Undo

Use `/manage-account` and click **Stop ignoring account in server** to restore indexing. Your messages will be re-indexed over time.
