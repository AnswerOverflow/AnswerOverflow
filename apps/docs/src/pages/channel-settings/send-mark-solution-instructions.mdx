## What it does

Sends a message automatically to **all new threads / posts** in a channel instructing users how to mark their question as solved.

![Discord message sent by the Answer Overflow bot containing an image telling users to open the context menu on a message and use the mark solution command](/mark-solution-instructions.png)

## Why you might want it

For users who are new to the server, this is helpful to let them know how to mark their question as solved. For users who are experienced, this is helpful to remind them to mark their question as solved.

## Enabling

Use `/channel-settings` in the channel you want to enable sending mark solution instructions in. Click the **{OPEN_HELP_CHANNEL_UTILITIES_LABEL}** menu and then click **{ENABLE_SEND_MARK_AS_SOLUTION_INSTRUCTIONS_LABEL}**

## Disabling

Use `/channel-settings` in the channel you want to disable sending mark solution instructions in. Click the **{OPEN_HELP_CHANNEL_UTILITIES_LABEL}** menu and then click **{DISABLE_SEND_MARK_AS_SOLUTION_INSTRUCTIONS_LABEL}**

Existing threads will not have the instructions removed from them.
