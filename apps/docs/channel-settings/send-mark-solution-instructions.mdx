## What it does

Automatically sends a message to every new thread or post showing users how to mark their question as solved.

![Discord message sent by the Answer Overflow bot containing an image telling users to open the context menu on a message and use the mark solution command](/images/mark-solution-instructions.png)

## Why use it

New users learn how to mark solutions. Experienced users get a reminder. Either way, more questions get marked as solved.
