## What it does

Marks new users to your server as consenting to [display their messages](/user-settings/displaying-messages) publicly on Answer Overflow.

## Enabling


Use the `/server-settings` slash command in any channel of your server and then click on the **Enable read the rules consent** button.

<Warning>
	You must have the following in your [membership
	screening](https://support.discord.com/hc/en-us/articles/1500000466882-Rules-Screening-FAQ)
	for this to work.

	`This server uses Answer Overflow to index content on the web. By posting in indexed channels, you agree to have your messages publicly displayed on Answer Overflow to help others find answers.`
</Warning>

## Disabling

Use the `/server-settings` slash command in any channel of your server and then click on the **Disable read the rules consent** button.

You can remove the consent prompt from your membership screening as it will no longer be applied to new members.
